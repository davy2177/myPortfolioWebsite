Student Donation Platform

I developed an innovative Student Donation platform aimed at connecting students in need with generous donors. This website, built using React, CSS, HTML and JavaScript, provide a secure and intuitive environment for facilitating donations and support

Key Features

  • User Authentication: Secure login and registration system ensuring user data privacy and security.
  • Donation Management: Easy-to-user interface for donors to browse and select student needs and for students to post their requirements.
  • Real-Time Updates: Live Updates on donation statuses and student needs to keep users informed.
  • Responsive Design: Ensures a seamlesss experience across all devices, from desktops to mobile phones.
  • Team Member Management: Robust admin panel for managing user roles and permissions.

This project demonstrates my expertise in creating secure, user-friendly web applications and my commitment to using technology to support and uplift communities.
